# Editor Console Commands

Quick reference for useful console commands in Alis.

---

## Collision

| Command | Description |
|---------|-------------|
| `collision.enable 1` | Enable collision processing |
| `collision.enable 0` | Disable collision processing |
| `show collision` | Toggle collision visualization |
| `p.VisualizeMovement 1` | Show movement/collision debug info |

---

## Debug

| Command | Description |
|---------|-------------|
| `stat fps` | Show FPS counter |
| `stat unit` | Show frame time breakdown |
| `stat game` | Show game thread stats |
| `log LogTemp VeryVerbose` | Set log category verbosity |

---

## Navigation

| Command | Description |
|---------|-------------|
| `show navigation` | Toggle navmesh visualization |
| `ai.debug 1` | Enable AI debugging |

---

## Useful Tips

- Open console: `~` (tilde) key
- List all commands starting with X: `help X`
- PIE and standalone use same command syntax
